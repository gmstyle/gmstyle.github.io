1-D: I miei dati sono protetti? Chi può accedere ai miei dati?

Sì, i tuoi dati sono protetti da un doppio sistema di protezione: prima di tutto, i tuoi account sono archiviati in un database crittografato che solo Password Security Storage può leggere e scrivere. Anche la password principale è crittografata e protegge il database. Solo tu puoi accedere ai tuoi dati con la tua password principale. Nessun dato viene condiviso al di fuori di Password Security Storage. Anche i backup dei dati sono compressi e protetti dalla password principale.

2-D: Ho perso la password dell' applicazione, cosa posso fare?

R: NIENTE, non c'è modo di recuperare la password.
Se perdi la password principale non potrai nemmeno ripristinare i backup precedenti.

3-D: Posso trasferire e ripristinare il mio backup sul mio nuovo dispositivo? E come?

R: SÌ! È possibile trasferire facilmente i dati utilizzando un file di backup che può essere ripristinato sul nuovo dispositivo UTILIZZANDO LA STESSA PASSWORD PRECEDENTE.
1- Trasferisci il file zip di backup denominato "encrypted-pss_backup-1-1-2020.zip" nella memoria interna del tuo nuovo dispositivo.
2- Installa Password Security Storage sul tuo nuovo dispositivo.
3- Imposta la STESSA PASSWORD PRECEDENTE , quella che avevi anche sul vecchio dispositivo e con la quale hai effettuato il backup che stai per ripristinare.
4- Da Password Security Storage vai in IMPOSTAZIONI e ripristina il backup utilizzando la funzione RIPRISTINA.

4-D: Posso recuperare i dati memorizzati in Password Security Storage dopo un ripristino del mio dispositivo?

R: Puoi recuperare i tuoi dati solo ripristinando un file di backup.
Se non hai creato un file di backup prima, i tuoi dati saranno persi e non recuperabili in altro modo.

5-D: Non riesco a ripristinare lo zip crittografato!

R: 1- Assicurati che il file di backup sia un file zip e abbia il nome tipo "encrypted-pss_backup-1-5-2020.zip".
2- Assicurati che la PASSWORD dell'applicazione sia la stessa con cui è stato creato il file di backup.

English


Password Security Storage is released for free and as is. No data stored in the app is made public or reaches any external website. Everything stored in the app is stored locally. Only Password Security Storage can read and write to the encrypted database. If you uninstall the app all data is deleted and destroyed. The developer is not responsible for any data loss or misuse of the app. Remember that losing or forgetting the master password will make the app and all previously created backups inaccessible. There is no way to reset the master password.

1-Q: My data are on secure? Who can access to my data?

Yes, your data are on secure by a double protection system: first of all, your accounts are storaged in a encrypted database that only Password Security Storage can read and write. The master password is also encrypted and protects the database. Only you can access to your data with your master password. No data are shared out of the Password Security Storage app. Data backups are also compressed and protected by your master password. Only you

2-Q: I lost my password, what can i do?

A: NOTHING, there is no way of recovering the app password.
If you loose your password you can't recover or restore your previous backups.

3-Q: Can i transfer my backup zip to my new device? And how?

A: YES! You can easily transfer your data using a backup file which can be restored to your new device USING THE SAME PREVIOUS PASSWORD.
1- Transfer the backup zip file named 'encrypted-pss_backup-1-1-2020.zip' to the internal memory of your new device.
2- Install Password Security Storage on your new device.
3- Set the SAME PREVIOUS PASSWORD, the one you also had on the old device and with which you made the backup you are about to restore.
4- From Password Security Storage SETTINGS restore the backup using the RESTORE function.

4-Q: How can i recover my data after the device reset?

A: You can only recover your data by restoring a backup file.
If you haven't created a backup file before then your data is lost.

5-Q: I can't restore the encrypted zip!

A: 1- Make sure the backup file is a zip file and has the name like 'encrypted-pss_backup-1-5-2020.zip'.
2- Make sure that the PASSWORD of the application is the same as the one with which the Backup file was created.

1-D: I miei dati sono protetti? Chi può accedere ai miei dati?

Sì, i tuoi dati sono protetti da un doppio sistema di protezione: prima di tutto, i tuoi account sono archiviati in un database crittografato che solo Password Security Storage può leggere e scrivere. Anche la password principale è crittografata e protegge il database. Solo tu puoi accedere ai tuoi dati con la tua password principale. Nessun dato viene condiviso al di fuori di Password Security Storage. Anche i backup dei dati sono compressi e protetti dalla password principale.

2-D: Ho perso la password dell' applicazione, cosa posso fare?

R: NIENTE, non c'è modo di recuperare la password.
Se perdi la password principale non potrai nemmeno ripristinare i backup precedenti.

3-D: Posso trasferire e ripristinare il mio backup sul mio nuovo dispositivo? E come?

R: SÌ! È possibile trasferire facilmente i dati utilizzando un file di backup che può essere ripristinato sul nuovo dispositivo UTILIZZANDO LA STESSA PASSWORD PRECEDENTE.
1- Trasferisci il file zip di backup denominato "encrypted-pss_backup-1-1-2020.zip" nella memoria interna del tuo nuovo dispositivo.
2- Installa Password Security Storage sul tuo nuovo dispositivo.
3- Imposta la STESSA PASSWORD PRECEDENTE , quella che avevi anche sul vecchio dispositivo e con la quale hai effettuato il backup che stai per ripristinare.
4- Da Password Security Storage vai in IMPOSTAZIONI e ripristina il backup utilizzando la funzione RIPRISTINA.

4-D: Posso recuperare i dati memorizzati in Password Security Storage dopo un ripristino del mio dispositivo?

R: Puoi recuperare i tuoi dati solo ripristinando un file di backup.
Se non hai creato un file di backup prima, i tuoi dati saranno persi e non recuperabili in altro modo.

5-D: Non riesco a ripristinare lo zip crittografato!

R: 1- Assicurati che il file di backup sia un file zip e abbia il nome tipo "encrypted-pss_backup-1-5-2020.zip".
2- Assicurati che la PASSWORD dell'applicazione sia la stessa con cui è stato creato il file di backup.
